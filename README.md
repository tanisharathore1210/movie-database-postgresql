# 🎬 MovieVerse: SQL-Based Movie Database Management & Analysis

## 📌 Project Overview

This project is a Movie Database Management System built using PostgreSQL. It demonstrates relational database design, data insertion, and advanced SQL querying techniques.

The database stores information about movies, actors, genres, users, and movie ratings. It also includes multiple analytical queries to retrieve meaningful insights from the data.

---

## 🛠 Technologies Used

- PostgreSQL
- SQL
---

## 📂 Database Schema

The project contains the following tables:

- Movies
- Genres
- Movie_Genres
- Actors
- Movie_Actors
- Users
- Ratings

These tables are connected using Primary Keys and Foreign Keys.

---

## 📊 Features

- Create relational database schema
- Insert sample movie data
- Manage movie genres
- Store actor information
- Store user ratings and reviews
- Perform SQL analysis queries

---

- Window Functions (RANK)

## 🔍 Sample Queries

- Top 5 Highest Rated Movies
- Movies Rated Above 9 by Indian Users
- Movies Count by Genre
- Most Featured Actor
- Average Rating Given by Each User
- Movie Release Ranking
- Movie Era Classification
